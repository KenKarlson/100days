Привет! Вот твоя задача на понедельник:

Упражнение “Работа с объектами и замыканиями”
Тебе нужно создать функцию createCounter, которая будет возвращать объект с тремя методами:
increment() - увеличивает значение на 1
decrement() - уменьшает значение на 1
getValue() - возвращает текущее значение
При создании счетчика нужно передать начальное значение.

Требования:

Значение счетчика должно быть приватным
Методы должны работать только с внутренним состоянием
При создании счетчика должно быть возможно передать любое число (включая отрицательные)
Если при создании счетчика не передано значение, использовать 0
